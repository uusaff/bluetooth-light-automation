## ⚠️ Important Note
**CAUTION:** This script currently only works with non-sequential LED lights.

## 📌 Overview
This project is a Python automation script designed to connect and control Bluetooth ambient lights directly from a laptop. Instead of relying on a mobile app, this script allows you to set the room's vibe, change colors, and manage the lights programmatically through the terminal.

## 🚀 Features
* **Seamless Connection:** Automatically discovers and connects to the paired Bluetooth LED lights.
* **Vibe Automation:** Quickly set custom lighting modes right from the laptop.
* **Lightweight & Fast:** Uses minimal system resources to send Bluetooth commands.
